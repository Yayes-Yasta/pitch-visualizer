# pitch-visualizer
This program takes in a midi file and compares the played notes to the microphone input to produce a rhythm game. This only works when one note is played at a time, which makes it useful for practicing singing but it has big limitations when using it to play multiple notes at a time.

### Usage
```
python main.py [midi_file_name]
```
Example:
```
python main.py demo.midi
```
